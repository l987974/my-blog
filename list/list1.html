第一数据在哪里，config.js和md里面头部里面，vue组件里面可以通过this.$site可以访问到。
第二页面怎么跳转，docs文件夹下的README.md看做首页，其他页面就是放在跟他同级的文件夹里面的md文件。 而md可以在头部设置layout,页面就变成了vue组件了。跳转用a标签跳转。
第三样式，.vuepress > styles 这里面是全局样式；vue组件里面的 style标签 加上属性 scoped就是局部样式，md里面也可以用style标签。

# vuepress使用步骤

# 1、全局安装vuepress

// 命令行
npm install -g vuepress

# 2、新建一个空文件夹docs,在docs里面新建一个文件README.md（相当于项目的首页）, 输入内容，之后命令行输入vuepress dev。

<!-- README.md -->
# hello world

// 命令行
vuepress dev

# 3、docs文件下新建一个文件夹.vuepress, .vuepress下面新建一个文件夹components（放vue组件）,再新建一个文件config.js,然后把docs里面的README.md的内容可以。
