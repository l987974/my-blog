# vuepress使用步骤

# 1、全局安装vuepress

// 命令行
npm install -g vuepress

# 2、新建一个空文件夹docs,在docs里面新建一个文件README.md（相当于项目的首页）, 输入内容，之后命令行输入vuepress dev

<!-- README.md -->
# hello world

// 命令行
vuepress dev

# 3、docs文件下新建一个文件夹.vuepress, .vuepress下面新建一个文件夹components,再新建一个文件config.js,然后把docs里面的README.md的内容可以使用vuepress文档
